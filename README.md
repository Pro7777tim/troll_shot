# TrollShot

## About

Browser game similar to Buckshot Roulette.
<br/>
In the game you have to play Russian roulette with a troll, using various objects.
<br/>
The game has complex mechanics (luck, money, achievements, different modes).
<br/>There are many jokes. The game is made very high quality.
<br/>
The animations are smooth, the textures and sounds are high quality.
<br/>
Management is not complicated. Have a luck game! ;>
<br/>
P.s don't anger the troll!

## Run

You can run it locally (run python server or open index.html), or you can use a public url.
